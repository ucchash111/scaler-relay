# Contributing to Scalar Relay

First off, thank you for considering contributing to Scalar Relay! It's people like you that make Scalar Relay such a great tool.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If you find a new bug, please open a new issue and provide as much detail as possible.

### Suggesting Enhancements
* Open an issue to discuss your idea before you start working on it.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and ensure they follow the code style.
4. Submit a pull request with a clear description of your changes.

## Code of Conduct
Please be respectful and helpful to others in the community.
