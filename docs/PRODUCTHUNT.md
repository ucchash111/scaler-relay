# 🚀 ProductHunt Launch Kit | Scalar Relay

## 📝 The Basics
- **Product Name**: Scalar Relay
- **Tagline**: The No-Compromise SMTP Bridge. Secure, stateless, and developer-first.
- **Description**: Scalar Relay is a professional HTTP-to-SMTP bridge that decouples sensitive credentials from your app logic. Built for developers who want the security of AWS SES with the simplicity of a local relay.

## 🏹 The Problem & Solution (Makers Comment)
"Hey ProductHunt! 👋

I built Scalar Relay because I was tired of scattering my SMTP credentials across dozens of Docker containers, n8n workflows, and CI/CD pipelines. Every time I changed a password, I had to update ten different places.

**Scalar Relay changes that.** It provides a single, secure gateway for all your outbound mail.

**What makes it special?**
- 🛰️ **Stateless Logic**: RAM-only logs mean your data is never persisted longer than needed.
- 🔐 **Multi-Tenancy**: Give every project its own API key without sharing the master SMTP pass.
- 🛠️ **Developer First**: 1-click install via Docker, Coolify support, and a beautiful Tailwind dashboard.
- 🏎️ **Stateless Speed**: Built with Node.js and Nodemailer for high-performance throughput.

It's open-source, lightweight, and ready for 2026. I'd love to hear your feedback on how we can make SMTP relaying even simpler!"
